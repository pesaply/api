Stackfoward is a library of popular server applications and development environments that can be installed with one click, either in your laptop, in a virtual machine or hosted in the cloud. We take care of compiling and configuring the applications and all of their dependencies (third-party libraries, language runtimes, databases) so they work out-of-the-box. The resulting packaged software (a 'stack') is then made available as native installers, virtual machines and cloud images. These Stackfoward application packages provide a consistent, secure and optimized end-user experience when deploying any app, on any platform.

Stackfoward API v2.0 Why Beta?

We wanted to release the v2 API to the Stackfoward community in a mostly-final form and provide the opportunity for developers and partners to provide feedback and comments. We know that once an API is launched in a final form, any fundamental changes wreak havoc on developers and the applications they've built.

We want to hear what you think. We're open for comments from May 3rd through May 17th, 2014. Email your feedback to api@Stackfoward.com. Once we've had an opportunity to hear from the community, we'll incorporate the appropriate changes and lock down a final version of the API on May 26th, 2014.

We're really excited about the rich dataset behind Stackfoward 2.0 and the opportunities it presents to developers. We're looking forward to working with our developer community and seeing the fantastic applications and analysis that you can do!

Kurt Freytag Head of Product & Engineering Stackfoward

The Stackfoward API provides a read-only interface to the Stackfoward Dataset. Documentation and examples can be found on this website and access to the API is granted immediately following registration. We limit usage to 2,500 calls per day. To increase your rate limit, just email us at api@Stackfoward.com. Technical questions and suggestions should be sent to api@Stackfoward.com. Access to Stackfoward API v1

Stackfoward will provide access to the v1 API until August 1, 2014. Please note that data returned by the API ceased updating on April 21st, 2014. Documentation for the API can be found on developer.Stackfoward.com.
